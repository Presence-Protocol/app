# Presence protocol

## Metadata format

To be able to show the POAP, the organizer needs to add a json file following the NFT format the chain is using.

### Collection
```json
{
  "name": "Alephium meetup",
  "description": "One of the many",
  "image": "<image file>"
}
```



### POAP

```json
{
  "name": "Alephium meetup",
  "image": "<image file>"
}
```
